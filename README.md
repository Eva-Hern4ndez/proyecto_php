# Proyecto PHP 

## Descripción: 
Página web en donde se puede visualizar la frase Hola mundo, seguido de nuestro nombre. Y se le adjunta una imagen generada por inteligencia artificial. 

## Prerequisitos:
1. instalar apache2
2. instalar php

## Pasos de Ejecución:
Podemos realizarla con la dirección IP de una máquina virtual de Ubuntu. En su defecto desde la IP de la máquina donde se esté trabajando el programa, seguido de la ubicación de las carpetas donde se guarde o donde se este trabajando el proyecto.

Ejemplo donde se puede visualizar el proyecto
http://192.168.1.105/proyecto_php/index.php
